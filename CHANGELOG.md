# Changelog

## re:Optimized Vanilla v1.0.0
**Minecraft:** 26.1.2

Initial public release of **re:Optimized Vanilla**.

### Added

#### Performance
- Integrated modern performance optimization mods.
- Improved FPS and overall game responsiveness.
- Reduced memory usage and loading times.

#### Visual Enhancements
- Improved clouds, particles, lighting, foliage, and environmental effects.
- Better entity rendering.
- Dynamic lights support.
- Optional shader support via Iris.

#### User Interface
- Redesigned main menu.
- Enhanced HUD and inventory interface.
- Improved menus and GUI elements.
- Better tooltips and item information.

#### Animations
- Added player animation improvements.
- Better first-person and third-person movement.
- Improved interaction animations.

#### Quality of Life
- Better inventory management.
- Better advancement screen.
- Better chat experience.
- Improved multiplayer experience.
- Better accessibility and usability.

#### Audio
- Enhanced ambient sounds.
- Improved sound physics and environmental audio.

### Features
- 100% client-side.
- Multiplayer friendly.
- No server installation required.
- Preserves vanilla gameplay mechanics.
- Preserves vanilla progression.
- Preserves vanilla mobs and items.
- Preserves vanilla balance.

## re:Optimized Vanilla v1.0.1
**Minecraft:** 26.1.2

This update focuses on restoring a completely vanilla world generation experience while improving overall compatibility.

### Removed
- Removed **Blooming Biosphere**.
  - Restored vanilla terrain generation.
  - Prevents non-vanilla mountain and terrain formations.

- Removed **Explorify**.
  - Removed all additional structures.
  - Vanilla structures now generate exactly as intended.

### Fixed
- Fixed non-vanilla world generation.
- Fixed Flat World generating incorrectly under certain circumstances.
- Improved compatibility with vanilla world presets.

### Improved
- Better consistency with the project's vanilla philosophy.
- Minor internal cleanup and optimization.
