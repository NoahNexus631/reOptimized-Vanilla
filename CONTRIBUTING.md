# Contributing

Thank you for your interest in improving re:Optimized Vanilla.

## Before opening an issue

Please:

- Make sure you're using the latest version.
- Check existing issues to avoid duplicates.
- Include your Minecraft version and modpack version.
- Attach crash reports or latest.log whenever possible.

## Feature Requests

Feature requests are welcome, but they should align with the philosophy of re:Optimized Vanilla:

- Vanilla-friendly
- Client-side
- Performance-focused
- Quality of Life
- Visual improvements

Mods that significantly alter gameplay are generally outside the scope of this project.
