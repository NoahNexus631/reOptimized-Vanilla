# re:Optimized Vanilla

A Fabric client-side modpack focused on improving Minecraft without changing its vanilla gameplay.

## Features

- Better Performance
- Better Visuals
- Better UI
- Better Animations
- Better Quality of Life
- Shader Support
- Multiplayer Friendly

## Minecraft Version

26.1.2

## Loader

Fabric
